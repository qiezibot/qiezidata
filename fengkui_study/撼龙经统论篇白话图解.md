# 《撼龙经·统论篇》白话翻译 + 图解

## 写在前面

《撼龙经》是杨筠松的代表作，讲的是怎么看山势龙脉。风水里说的"龙"不是真龙，**就是山脉**。"撼龙"就是"看懂山脉走势"。

统论篇是全书总纲，把核心概念先讲清楚。下面逐句翻译 + 图解说明。

---

## 【原文】须猕山是天地骨，中镇天地为巨物。如人背脊与项梁，生出四肢龙突兀。

### 白话翻译
昆仑山（须猕山）是天地的脊梁骨，像一根巨大的柱子撑在天地中间。就像人的背脊连着脖子，从这条脊梁骨上生出四肢——山脉也是这样，从主干分出支脉。

### 图解

```
         ★ 昆仑山（天地之骨）
              |
          ●━━━┻━━━●  （主山脉，像人的背脊）
         /    |    \
        /     |     \
   支脉1  支脉2   支脉3  （像四肢延伸出去）
```

**💡 核心概念：** 风水认为天下所有山脉都起源于昆仑山（须猕山），它是"祖山"。

---

## 【原文】四肢分出四世界，南北西东为四派。西北崆峒数万程，东入三韩隔杳冥。唯有南龙入中国，胎宗孕祖来奇特。

### 白话翻译
四条大山脉从昆仑向四个方向延伸，分成四大区域。西北的崆峒山绵延几万里，东边进入朝鲜半岛隔着大海。只有南边的龙脉进入了中国内地，这条龙最有灵气、最奇特。

### 图解

```
                昆仑
               ┌┴┐
    北龙←─────┤ ├─────→东龙（入朝鲜/日本）
               │ │
    西龙←─────┤ ├─────→南龙 ★★★★★ 进入中国！
               └─┘
                  ↓
              中国内地
            （风水宝地主要在这条龙上）
```

**💡 核心概念：** 天下龙脉分三支——北龙（西北）、中龙（中原）、南龙（江南）。最贵重的是**南龙**，因为它进入了中国腹地，孕育了最多帝王将相。

---

## 【原文】黄河九曲为大肠，川江屈曲为膀胱。分肢劈脉纵横去，气血勾连逢水住。

### 白话翻译
黄河弯弯曲曲像人的大肠，长江弯弯曲曲像人的膀胱。山脉像血管一样纵横交错，遇水就停住（水界气止）。

### 图解

```
    ═══ 山脉（龙） ═══
    ~~~~ 河流（水） ~~~~

            龙脉
        ════════════
       ╱            ╲
      ╱    ~~~~~~~    ╲  ← 长江（膀胱）
     ╱   ~黄河~     ╲
    ╱  （大肠）      ╲
   ╱                  ╲
  ╱                    ╲
 ●━━━━━━━━━━━━━━━━━━━━●
 山脚                   山脚
```

**💡 核心概念：** 风水把大地比作人体。山脉=骨骼=龙；河流=血管/肠胃=水。山遇水而止，水把山脉的"气"截住、聚住，这就是吉祥的地方。

## 【原文】高山大垄峰峦秀，不唯江南多如此。南龙一干亦多角，五岭以上多峰峦。

### 白话翻译
高山大脉、峰峦秀丽的地方，不只是江南才有。南龙这一支也有很多分支，五岭以南也有很多秀丽的山峰。

---

## 【原文】大率行龙有真星，星峰磊落是音身。高山须认星峰起，平地龙行别有名。有星不可使无脱，脱处有星亦合格。

### 白话翻译
凡是真正的龙脉，山体上一定有"星峰"（突出的大石头/山峰），星峰一个一个排列清楚，这才是真龙。高山要认山峰，平地的龙行则叫"平洋龙"，另有一套说法。

### 图解

**真龙（有星峰）：**

```
       ★       ★       ★       ★
      / \     / \     / \     / \
     /   \   /   \   /   \   /   \
    /     \ /     \ /     \ /     \
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 星峰+星峰+星峰+星峰 → 真龙！

```

**假龙（没有星峰）：**

```
    ╱╲    ╱╲    ╱╲
   ╱  ╲  ╱  ╲  ╱  ╲     ← 光秃秃，没有突出山头
  ╱    ╲╱    ╲╱    ╲
 ────────────────────
  这就不行 ❌
```

**💡 核心概念：** "星"就是山的形状。不同的形状对应不同的吉凶——贪狼（尖）、巨门（方）、武曲（圆）是吉星，破军（破碎）、廉贞（尖削带火）是凶星。

---

## 【原文】大抵山形虽在地，其气通天本一理。高山平地总一般，只是深浅高低异。

### 白话翻译
总的来说，山虽然在地面上，但它的"气"是和天相通的。高山和平地的道理是一样的，只是深浅高低不同而已。

## 【原文】识得真龙与假龙，方知吉凶分途路。真龙身上有正形，假龙身上皆斜曲。真龙落处气必聚，假龙落处气不聚。

### 白话翻译
能分辨真龙和假龙，才能分清楚吉和凶。真龙的山体端正整齐，假龙的山体歪斜弯曲。真龙落脉的地方，气是聚在一起的；假龙落脉的地方，气是散的。

### 图解

**真龙：气聚（风水好 ✅）**

```
  → 适合做穴位（坟/宅）
```

## 【原文】高山大垄出龙楼，平地山岗出殿阁。楼殿之上寻堂局，堂局之内看凹凸。

### 白话翻译
高山大脉会产生"龙楼"（像高楼一样的山形），平地的山岗会产生"殿阁"（像宫殿一样的低山）。在楼殿之上要找"堂局"（开阔平坦的地形），在堂局里面要看凹凸（凸的地方是"穴"，凹的地方是"水口"）。

### 图解

```
         龙楼（高山）
          ╱|\╲
         ╱ | \╲
        ╱  |  \╲
       ╱   |   \╲
      ╱    |    \╲
     ╱ 堂局│开阔 │╲   ← 找平坦开阔处
    ╱      |      ╲
   ╱   ●穴 | ●穴   ╲  ← 凸出点=穴位
  ╱        |        ╲
 ━━━━━━━━━━━━━━━━━━━━
  ~~~~ 水口（水流出口）~~~~
```

**💡 核心概念：** "堂局"就像房子的客厅，要开阔平坦。"穴"是堂局里微微凸起的地方，是点穴位置。

## 【原文】寻龙须寻祖与宗，不辨祖宗何足语。大山之下寻小山，小山之下有平地。欲知富贵与贫贱，须向穴中审堂气。

### 白话翻译
寻龙必须找到它的"祖宗"（源头山脉），连源头都不清楚那还谈什么。大山下面找小山，小山下面找平地。要知道这个地方是富贵还是贫贱，要在穴的位置看"堂气"（周围的环境气场）。

### 图解

```
         ★ 祖山（昆仑/最高山）
          │
          │
         ★ 宗山（主峰）
         ╱|╲
        ╱ | ╲
       ╱  |  ╲
      ★   ★   ★ 父母山
     ╱|╲ ╱|╲ ╱|╲
    ★ ★ ★ ★ ★ ★ ★ 子孙山
     ╲│  │╱
      ●穴

从祖→宗→父母→子孙，一气连贯
```

## 【原文】高山须认星峰起，平地龙行别有名。峰以星名星有九，九星之外无他求。

### 白话翻译
高山要看星峰，平地龙有另一种叫法。山峰用九种星来命名，九星之外没有别的分类了。

## 【统论篇总结】

一句话概括《撼龙经统论篇》：

**天下龙脉出昆仑，认星识峰辨假真。高山平地都有龙，气聚之处是穴珍。**

白话版就是：
1. 所有山脉都从昆仑山发源
2. 要看山脉有没有漂亮的"星峰"（山头形状好）
3. 真龙气聚，假龙气散
4. 气聚的地方就是风水宝地，可以点穴
