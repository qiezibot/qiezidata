### pyJianYingDraft/pyCapCut（替代方案）
- GitHub上有**第三方Python库**可以直接生成剪映草稿文件
- **pyJianYingDraft**（3300⭐）：生成剪映草稿JSON，支持视频/音频/文本/贴纸/特效/滤镜/转场/关键帧/动画/蒙版
  - 安装：`pip install pyJianYingDraft`
  - 草稿生成功能**支持剪映5+所有版本**（README说可以）
  - 带有**JianyingController**支持自动导出（通过uiautomation操作UI控件）
  - 安装时带uiautomation和comtypes依赖
- **pyCapCut**（508⭐）：从pyJianYingDraft迁移的CapCut版本
  - 安装：`pip install pycapcut`
  - API与pyJianYingDraft几乎一样

### 2026-05-25 15:50 实测结果（重要）
**pyJianYingDraft 在剪映10.6上有效性测试：**
1. ✅ `DraftFolder.create_draft("name", 720, 1280)` → 成功生成`draft_content.json`和`draft_meta_info.json`
2. ✅ `ScriptFile`添加视频、音频、文本轨道成功（三段视频+音频+三行字幕）
3. ❌ 剪映10.6**不对未加密JSON做兼容**
   - 剪映10.6真实草稿：draft_content.json是**Base64编码加密数据**（解码后也不是JSON）
   - pyJianYingDraft生成的：明文JSON（剪映首页**不显示**）
4. ❌ `JianyingController.export_draft("test_api5")` → `DraftNotFound`（首页找不到草稿）
5. ❌ `DraftFolder.duplicate_as_template("5月25日", "新草稿")` → 加载加密JSON失败
6. ❌ pyJianYingDraft**无解密功能**（不提供decrypt方法）

**结论：pyJianYingDraft无法直接在剪映10.6上使用，因为剪映10.6的草稿使用Base64加密存储。**
