## 声纹识别系统大升级（v2 → v4）
### v2 问题
- 简单MFCC 39维特征，爹和姐分不清

### v4 改进（当前在线版本）
- 使用 **566维多维度特征**：MFCC(40维+delta+delta2) × 4种统计量 + F0基频 + 频谱特征 + 梅尔频谱 + 能量包络 + 共振峰近似
- 使用综合评分：cos相似度×0.5 + 样本平均×0.3 + 样本最低×0.2
- 独立的profile目录：`voice_profiles_v4/`
- **当前注册**：姐(4段)、爹(5段)
- 识别难度：爹和姐声线较接近，某些短句得分差小于0.01
