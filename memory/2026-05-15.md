### 核心问题：价格显示为"私聊"
- 之前发布的8个商品，所有价格在App上都显示为"私聊"，不是实际价格
- **根本原因**：闲鱼对电子资料/软件分类，强制要求选"计价方式"字段（元/次、元/时、其他等）
- 之前采用"跳过弹窗"的方式发布，导致系统未记录计价方式 → 自动将价格变为"私聊"
- 首次发布绕过流程图：
  - 传图→填描述→填价格→点发布 → 弹窗"你还没有填写价格"→ 点"不填继续发布" → 发布成功但价格变"私聊"
- **正确流程**：必须先选计价方式（元/次）→ 填价格 → 发布 → 价格正常显示

### 分类与计价方式的对应关系
- 分类"软件安装包/序列号/激活码" → 需选品牌+计价方式（选项仅"其他"）
- 分类"DeepSeek服务"（AI自动识别） → 需选计价方式（元/次、元/时）+ 服务类型
- 分类"互联网产品与运营技能培训" → 需选计价方式（元/次、元/时、元/课）
